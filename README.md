Welcome to the **Advent of Code** repository! This repository contains solutions to the Advent of Code puzzles implemented in Python. It's a fun and educational journey into problem-solving and algorithm design.

## About Advent of Code

[Advent of Code](https://adventofcode.com/) is an annual series of programming puzzles released daily in December. Participants solve small, self-contained puzzles to help Santa save Christmas!

## Repository Contents

This repository is organized as follows:

- **Yearly Directories**: Each top-level directory corresponds to a specific year (e.g., `2015`, `2025`).
- **Daily Challenges**: Each daily challenge is in its respective folder (e.g., `Day01`). Each folder includes:
  - `README.md` with the problem description.
  - Solution implemented in Python.
